xMsg and Clara-4 Java documents can be accessed following the links below.
xMsg is a general purpose publish-subscribe messaging system that assumes asynchronous nature of the communication between message producer and the consumer. However, xMsg API also includes a number of synchronous messaging mechanism, where it guarantees some level of direct communication between producer (publisher) and a consumer (subscriber).
